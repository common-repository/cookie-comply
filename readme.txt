== Description ==

Adds a message to your blog that says your visitors must to comply with the uk law and enable the cookies. Visitors must agree to use their cookies.

== Installation ==

1. Upload the files to `/wp-content/plugins/` directory
2. Activate the plugin through 'Plugins' menu in WordPress
3. Configure the text through the 'Settings' menu under 'Cookie Comply'
